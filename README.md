# Muhammad — AI Systems Engineer Portfolio

Premium Next.js + Tailwind portfolio showcasing backend, AI and automation case studies.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev

```bash
npm run dev
```

Deploy to Vercel — see `vercel-deploy.md` for recommended settings.
